# Scope and Non-Scope

## Scope

This skill pack targets the full middle pipeline of SIGMOD/VLDB paper production:
- project inventory
- novelty positioning
- writing
- method/proof drafting
- experiment design and auditing
- figure planning
- formatting
- reader/reviewer preflight

## Non-scope

- project-specific contribution constraints (for example, GLI/KNN-specific boundaries)
- venue-specific submission scripts or institutional policies not included in repository context
