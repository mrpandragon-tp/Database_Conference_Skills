# Review Rubric

Dimensions:
- novelty
- technical soundness
- proof soundness
- experiment quality
- baseline fairness
- reproducibility
- clarity and organization

Blocking consistency checks:
- title/Abstract/Introduction/contribution identity mismatch
- contradiction between problem definition and algorithm behavior
- same concept renamed without definition
- guarantee depends on an unstated assumption
- core claim has no main-paper evidence
- experiment attribution points to systems engineering while novelty claims learning/algorithm design

Severity:
- fatal / major / moderate / minor

Reporting discipline:
- list findings in descending severity
- cite the exact section, equation, algorithm, figure, or paragraph
- map each fatal/major issue to a rewrite, proof, experiment, or figure artifact
- omit preference-only edits unless they cause reader misunderstanding
