# Reader Questions

- what is the exact contribution in one sentence?
- is that sentence about the same artifact named in the title?
- why are existing methods insufficient?
- does the Introduction state the main failure in a topic sentence?
- when the method first appears, do I know what it is and how it works?
- can I explain the data path after reading Figure 1?
- what is guaranteed vs empirically observed?
- which component improves performance and which establishes correctness?
- how do updates change physical and logical index state?
- what are the operating boundaries and limitations?
- which figure/table/theorem supports each major claim?
- does the ablation prove that the claimed learned/algorithmic component matters?
- are baselines assigned fair and understandable roles?
- are citations attached to the exact methods and claims they support?
- are concepts renamed between Abstract, Introduction, Method, algorithms, and proofs?
