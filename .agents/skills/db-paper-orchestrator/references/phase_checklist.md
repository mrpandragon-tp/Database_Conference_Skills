# Phase Checklist

1. Project inventory
- code, experiments, logs, result tables, draft assets
- reference papers, venue rules, page budget, and appendix restrictions
- local/GitHub/Overleaf source-of-truth state
- portable project instructions and context-resilient state notes

2. Novelty and positioning
- one-sentence paper identity
- claim-prior map
- scope and boundary statement

3. Evidence planning
- experiment matrix
- proof obligation list
- figure plan

4. Drafting
- Abstract/Intro/Method/Experiments/Conclusion
- claim-evidence map
- reverse outline and terminology ledger

5. Technical validation
- proof soundness pass
- experiment fairness pass

6. Reader/reviewer pass
- rejection-risk report
- fix plan
- contradiction, renamed-concept, and serious-language audit

7. Format and final polish
- layout, references, consistency, PDF checks
- annotated-review preservation and remote-head verification
