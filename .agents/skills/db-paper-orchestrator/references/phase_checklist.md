# Phase Checklist

1. Project inventory
- code, experiments, logs, result tables, draft assets

2. Novelty and positioning
- claim-prior map
- scope and boundary statement

3. Evidence planning
- experiment matrix
- proof obligation list
- figure plan

4. Drafting
- Abstract/Intro/Method/Experiments/Conclusion
- claim-evidence map

5. Technical validation
- proof soundness pass
- experiment fairness pass

6. Reader/reviewer pass
- rejection-risk report
- fix plan

7. Format and final polish
- layout, references, consistency, PDF checks
