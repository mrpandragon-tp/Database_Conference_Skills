# Visual Style Rules

- clean academic style
- readable fonts and consistent sizing
- limited color palette with clear baseline differentiation
- stable color or marker for the proposed method across experiment families
- avoid visual clutter and heavy effects
- remove dense minor-grid lines; retain only scale-significant guides when useful
- use logarithmic axes for order-of-magnitude gaps
- avoid horizontal bars unless labels or aspect ratio make them superior
- scale figures so labels are close to body-text size in the final PDF
- prefer single-column width for compact panels; use double-column width only when necessary
- export vector format when possible
