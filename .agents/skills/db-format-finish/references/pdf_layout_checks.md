# PDF Layout Checks

- figure/table captions readable
- no clipped labels
- figure text close to body-text size after scaling
- single- and double-column choices justified by information density
- floats placed near their first meaningful discussion
- balanced page breaks
- no orphan headings, isolated words, or single-line paragraph fragments
- legible equations
- no excessive whitespace below figures, tables, or displays
- consistent margins and spacing
- first bibliography page and main-paper end page recorded separately
- affected pages rendered and visually inspected after meaningful changes
